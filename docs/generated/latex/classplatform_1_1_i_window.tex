\doxysubsection{Detailed Description}
Window Interface. 

Independeant window system. Render backends can query native handles through \doxylink{structplatform_1_1_i_window_1_1_native_window_handle}{Native\+Window\+Handle} to create API-\/specific surfaces/\+swapchains.
